Create Flask routes for Loki connection management (connect, status, disconnect) and register the blueprint in main_compute.py.

Purpose: These routes expose the LokiClient to the frontend and API consumers. The connect route validates credentials via the two-step check, stores them in Vault. **Helper -- `_normalize_base_url(raw_url: str) -> Optional[str]`:**
Same logic as `GrafanaClient.normalize_base_url` but as a module-level function. Accept both `http://` and `https://` schemes (Loki is often internal/HTTP unlike Grafana which is always HTTPS):
- Strip whitespace
- If no scheme, prepend `https://`
- Strip trailing `/`
- Validate with regex: `^https?://[A-Za-z0-9._:-]+(\/.*)?$` (note: allows both http and https, and includes `:` for port numbers like `:3100`)
- Return normalized URL or None if invalid

**Helper -- `_get_stored_loki_credentials(user_id: str) -> Optional[Dict[str, Any]]`:**
Same pattern as `_get_stored_grafana_credentials`:
```python
def _get_stored_loki_credentials(user_id: str) -> Optional[Dict[str, Any]]:
    try:
        return get_token_data(user_id, "loki")
    except Exception as exc:
        logger.error(f"Failed to retrieve Loki credentials for user {user_id}: {exc}")
        return None
```

**Route 1: `POST /connect`** (implements CONN-01, CONN-02, CONN-03, CONN-04, CONN-05)
Decorator: `@require_permission("connectors", "write")`

Request body JSON fields:
- `baseUrl` (required) -- Loki instance URL
- `authType` (required) -- one of `"bearer"`, `"basic"`, `"none"`
- `token` (optional) -- for bearer auth
- `username` (optional) -- for basic auth
- `password` (optional) -- for basic auth
- `tenantId` (optional) -- X-Scope-OrgID for multi-tenant

Validation logic:
1. Parse JSON body: `data = request.get_json(force=True, silent=True) or {}`
2. Extract and validate `baseUrl` via `_normalize_base_url`. Return 400 if invalid: `{"error": "A valid Loki base URL is required (e.g., https://loki.example.com:3100)"}`
3. Extract `auth_type = data.get("authType", "none").strip().lower()`. Validate it is one of `("bearer", "basic", "none")`. Return 400 if invalid: `{"error": "authType must be one of: bearer, basic, none"}`
4. If `auth_type == "bearer"`: extract `token = (data.get("token") or "").strip()`. Return 400 if empty: `{"error": "API token is required for Bearer authentication"}`
5. If `auth_type == "basic"`: extract `username = (data.get("username") or "").strip()` and `password = (data.get("password") or "").strip()`. Return 400 if either empty: `{"error": "Username and password are required for Basic authentication"}`
6. Extract `tenant_id = (data.get("tenantId") or "").strip() or None`

Connection validation:
7. Log: `logger.info(f"[LOKI] Connecting user {user_id} to {base_url} (auth={auth_type})")`. If token exists, also log `mask_credential_value(token)`.
8. Instantiate `client = LokiClient(base_url, auth_type=auth_type, token=token, username=username, password=password, tenant_id=tenant_id)`
9. Call `result = client.test_connection()` in try/except `LokiAPIError as exc`. On error, return 502: `{"error": f"Failed to validate Loki connection: {exc}"}`
10. Extract `labels_count = len(result.get("labels", []))` for metadata.

Credential storage:
11. Build `token_payload`:
```python
token_payload = {
    "auth_type": auth_type,
    "base_url": base_url,
    "tenant_id": tenant_id,
}
if auth_type == "bearer":
    token_payload["token"] = token
elif auth_type == "basic":
    token_payload["username"] = username
    token_payload["password"] = password
```
12. Call `store_tokens_in_db(user_id, token_payload, "loki")` in try/except. On error, return 500: `{"error": "Failed to store Loki credentials"}`
13. Log success: `logger.info(f"[LOKI] Stored credentials for user {user_id} (auth={auth_type}, labels={labels_count})")`

Success response (200):
```python
return jsonify({
    "success": True,
    "baseUrl": base_url,
    "authType": auth_type,
    "tenantId": tenant_id,
    "labelsCount": labels_count,
})
```

**Route 2: `GET /status`** (implements CONN-06)
Decorator: `@require_permission("connectors", "read")`

1. Call `creds = _get_stored_loki_credentials(user_id)`
2. If not creds: return `{"connected": False}`
3. Extract `base_url = creds.get("base_url")`, `auth_type = creds.get("auth_type", "none")`
4. If not base_url: return `{"connected": False}`
5. Return:
```python
return jsonify({
    "connected": True,
    "baseUrl": base_url,
    "authType": auth_type,
    "tenantId": creds.get("tenant_id"),
})
```

**Route 3: `POST|DELETE /disconnect`** (implements CONN-07)
Decorator: `@require_permission("connectors", "write")`
Methods: `["POST", "DELETE", "OPTIONS"]`

Follow the Grafana disconnect pattern exactly:
1. Call `success, deleted_count = delete_user_secret(user_id, "loki")` in try/except
2. If not success: return 500 `{"success": False, "error": "Failed to delete stored credentials"}`
3. Log: `logger.info("[LOKI] Disconnected provider (deleted %s token entries)", deleted_count)`
4. Return 200: `{"success": True, "message": "Loki disconnected successfully", "deleted": deleted_count}`
5. On exception: return 500 `{"error": "Failed to disconnect Loki"}`
