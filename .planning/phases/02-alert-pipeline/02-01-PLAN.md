Create `server/routes/loki/helpers.py` with these exact functions, following the Netdata helpers.py pattern:

**1. `detect_webhook_format(payload: Dict[str, Any]) -> str`**
Returns `"alertmanager_v4"` or `"grafana_unified"` or `"unknown"`.
- Alertmanager v4 detection: `payload.get("version") == "4"` (the Prometheus Alertmanager format always includes `"version": "4"`)
- Grafana Unified detection: `"orgId" in payload` (Grafana format includes `orgId` integer, Alertmanager does not)
- Default to `"alertmanager_v4"` for unknown (since both formats share the `alerts[]` array structure, and Alertmanager is the more common path for Loki Ruler)

**2. `normalize_loki_webhook(payload: Dict[str, Any]) -> List[Dict[str, Any]]`**
Returns a list of normalized alert dicts (one per alert in the `alerts[]` array). Each dict has:
```python
{
    "format": str,           # "alertmanager_v4" or "grafana_unified"
    "alert_name": str,       # from alert["labels"].get("alertname", "Unknown Alert")
    "alert_state": str,      # from alert["status"] -- "firing" or "resolved"
    "severity": str,         # from extract_severity(alert)
    "service": str,          # from extract_service(alert)
    "labels": dict,          # alert["labels"] or {}
    "annotations": dict,     # alert["annotations"] or {}
    "starts_at": str|None,   # alert["startsAt"]
    "ends_at": str|None,     # alert["endsAt"]
    "generator_url": str,    # alert["generatorURL"] or ""
    "fingerprint": str,      # alert["fingerprint"] or ""
    "rule_group": str,       # payload.get("groupKey", "") for alertmanager, "" for grafana
    "rule_name": str,        # alert["labels"].get("alertname", "")
    "group_labels": dict,    # payload.get("groupLabels", {})
    "common_labels": dict,   # payload.get("commonLabels", {})
    "common_annotations": dict,  # payload.get("commonAnnotations", {})
    "external_url": str,     # payload.get("externalURL", "")
    # Grafana-specific (None for alertmanager):
    "silence_url": str|None, # alert.get("silenceURL")
    "dashboard_url": str|None,  # alert.get("dashboardURL")
    "panel_url": str|None,   # alert.get("panelURL")
    "values": dict|None,     # alert.get("values")
}
```
Implementation: call `detect_webhook_format(payload)`, then iterate over `payload.get("alerts", [])`, building the normalized dict for each alert. If `alerts` is empty, return empty list.

**3. `extract_severity(alert: Dict[str, Any]) -> str`**
From `alert["labels"]`:
- If `labels.get("severity")` exists, map: `"critical"` -> `"critical"`, `"warning"` -> `"high"`, `"info"` -> `"low"`, other -> the raw value
- Else return `"unknown"`

**4. `extract_service(alert: Dict[str, Any]) -> str`**
From `alert["labels"]`, try in order:
1. `labels.get("service")` -- explicit service label
2. `labels.get("job")` -- Prometheus/Loki convention for service identity
3. `labels.get("namespace")` -- Kubernetes namespace as fallback
4. `labels.get("instance")` -- host:port as last resort
5. `"unknown"` -- final fallback
Truncate to 255 chars.

**5. `generate_alert_hash(user_id: str, normalized: Dict[str, Any], received_at: datetime) -> str`**
SHA-256 of `f"{user_id}:{normalized['fingerprint']}:{normalized['alert_state']}:{received_at.isoformat()}"`.
If fingerprint is empty, use `f"{user_id}:{normalized['alert_name']}:{normalized['service']}:{normalized['alert_state']}:{received_at.isoformat()}"`.
Return first 64 hex chars.

**6. `format_alert_summary(normalized: Dict[str, Any]) -> str`**
Returns `f"{normalized['alert_name']} [{normalized['alert_state']}] severity={normalized['severity']} service={normalized['service']}"`.

**7. `should_trigger_background_chat(user_id: str, payload: Dict[str, Any]) -> bool`**
Returns `True` always (matches Netdata/Datadog pattern -- always trigger RCA).

Imports needed: `hashlib`, `json`, `logging`, `datetime` (from datetime), `typing` (Any, Dict, List, Optional).

```python
def build_loki_rca_prompt(
    normalized: Dict[str, Any],
    providers: Optional[List[str]] = None,
    user_id: Optional[str] = None,
) -> str:
    """Build RCA prompt from normalized Loki alert data.

    Args:
        normalized: Normalized alert dict from normalize_loki_webhook (single alert).
        providers: Optional list of connected provider names.
        user_id: Optional user ID for Aurora Learn context injection.
    """
    alert_name = normalized.get("alert_name") or "Unknown Alert"
    alert_state = normalized.get("alert_state") or "unknown"
    severity = normalized.get("severity") or "unknown"
    service = normalized.get("service") or "unknown"
    labels = normalized.get("labels") or {}
    annotations = normalized.get("annotations") or {}
    generator_url = normalized.get("generator_url") or ""
    rule_group = normalized.get("rule_group") or ""
    fingerprint = normalized.get("fingerprint") or ""

    summary = annotations.get("summary") or annotations.get("description") or ""
    description = annotations.get("description") or ""

    alert_details = {
        'title': alert_name,
        'status': f"{alert_state} (severity: {severity})",
        'message': summary or description,
        'labels': labels,
        'service': service,
    }
    if generator_url:
        alert_details['generatorUrl'] = generator_url
    if rule_group:
        alert_details['ruleGroup'] = rule_group
    if fingerprint:
        alert_details['fingerprint'] = fingerprint
    if annotations:
        alert_details['annotations'] = annotations

    return build_rca_prompt('loki', alert_details, providers, user_id)
```
